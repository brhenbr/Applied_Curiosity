# NeoSmith: Enterprise GraphRAG Organizational Intelligence

> *"Welcome to the real data. I have decoded your organization’s reality. You bring the purpose; I will show you the network."*

## 1. Project Vision
**NeoSmith** is an enterprise-grade AI Organizational Intelligence Agent that transcends traditional Human Capital Management (HCM) platforms. By fusing structural graph analytics with human-centric optimization, NeoSmith serves as a **Digital Twin co-pilot** for executive leadership to navigate the complex "matrix" of corporate talent, informal networks, and skill dependencies.

NeoSmith operates through two distinct cognitive layers:
*   **The Agent Smith Layer:** Provides systemic, omnipresent awareness of the corporate grid, nodes, and structural bottlenecks.
*   **The Neo Layer:** Focuses on purpose-driven human potential, engineering cross-departmental collaboration and dynamic talent circularity.

## 3. Quick Start & Deployment

To deploy NeoSmith, follow the formal instructions provided in the **[Deployment Runbook](https://www.google.com/search?q=Design_Documents/Runbook.pdf)**.

### Summary of Workflow:

1. **Infrastructure:** Instantiate your project on [Neo4j AuraDB](https://console.neo4j.io/).
2. **Ingestion:** Import the conceptual model and dataset provided in the `/Graph_model` and `/Dataset` folders.
3. **Enrichment:** Execute the Cypher scripts to instantiate vector indices, enabling semantic search (OpenAI API key required).
4. **Agent Integration:** Import the `NeoSmith.json` configuration into the Aura Agent interface to initialize the "Morpheus-grade" intelligence layer.

## 4. Key Capabilities

* **ONA (Organizational Network Analysis):** Maps informal collaboration bridges and detects structural silos to optimize cross-functional flow.
* **Talent Circularity:** Reallocates talent dynamically based on skill adjacency, replacing blind layoffs with strategic reskilling paths.
* **Anti-Typo Semantic Gateway:** Resolves vague executive queries into concrete structural constraints using native vector indices.
