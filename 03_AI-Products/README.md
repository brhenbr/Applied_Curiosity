# 03_AI-Products

**Applied Intelligence: From napkin sketches to autonomous (smart) stuffs.**

## About this section

If the core repository is about the *method*, this directory is about the *implementation*. Here, I translate the curiosity of "what if" into functional AI systems. 

This space collects my experiments in applied AI, ranging from autonomous agents that handle specific tasks to exploratory pipelines and custom tools. 

## Categories

*   **Agents:** Autonomous entities designed to solve multi-step problems.
*   **Pipelines:** Data workflows for RAG (Retrieval-Augmented Generation), ingestion and analysis.
*   **Playgrounds:** Rough sketches and "what-if" scenarios. Exploratory code that tests the boundaries of current LLM capabilities.
*   **Tooling:** Custom utilities built to optimize research and development processes.

*Status: Living workspace. Some projects are stable, others are highly experimental.*
