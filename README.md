# Applied_Curiosity Repo

> **Research, Development and Innovation (RD&I) in diverse contexts (academia, the office, or at the bar with friends).**

## About this Repository

For me, science is a process of discovery and learning, regardless of the setting. Whether I'm solving a complex problem for a corporate client, writing an academic paper or sketching ideas on a napkin, the method remains the same.

This repository collects the **Prototypes, Proofs of Concept (PoCs), and experiments** generated throughout this journey.

## What you'll find here

This codebase is a mix of formal and informal research, categorized not by job title, but by the problem being solved:

* **Academic:** Strict methodology, formal papers support and theoretical models.
* **Corporate:** Applied science, scalability tests and business-oriented solutions.
* **Personal/Experimental:** "What if" scenarios, napkin ideas turned into code and exploratory learning.

*Disclaimer: Some projects are polished, others are rough sketches. All are part of the process.*
